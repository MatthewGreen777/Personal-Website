About

This Discord bot was created to help manage and improve communication within the UMBC Tabletop Gaming Discord server. It was built to automate repetitive tasks, share announcements, organize events, and provide useful utilities that make it easier for members to stay engaged and informed. Whether it's sending reminders, managing roles, or providing quick access to club resources, the bot is designed to make the club's online space more interactive and efficient for both members and administrators.

Bot Commands

    • add-upcoming-events
      Allows people with the "Club Officer" role to add events.
    • remove-upcoming-event
      Allows officers to remove events by name.
    • song-of-the-day
      Lets officers post a song at a scheduled time.
    • club-games
      Displays a list of club-owned games.
    • list-member-games
      Shows a member's games if set up.
    • price-is-right
      Guess the MSRP of a board game.
    • random-fact
      Returns a random curated fact.
    • rock-paper-scissors
      Play against the bot.
    • roll-dice
      Rolls multiple dice and analyzes results.
    • roll-die
      Rolls a single die.
    • search-board-game
      Get info on board games.
    • server
      Shows current server population.
    • set-owned-games
      Link your BoardGameGeek account.
    • upcoming-events
      List future events.
    • user-most-played
      See your most played games.
